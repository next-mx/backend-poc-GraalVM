# Javascript NodeJS Service

## Instrucciones
```bash
cd js-node-service
```


#### NodeJS
```bash
which node  # Verifica que no sea la versión de GraalVM
node --version  # Verifica que sea la versión 10.18.1
npm install
npm pack   # Para medir el tamaño del componente
npm start 
```

#### GraalVM
```bash
~/.sdkman/candidates/java/19.2.1-grl/bin/npm install
~/.sdkman/candidates/java/19.2.1-grl/bin/npm pack  # Para medir el tamaño del componente
~/.sdkman/candidates/java/19.2.1-grl/bin/npm start
```
